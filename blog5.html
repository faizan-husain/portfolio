Blog#5 - How to integrate SQL Server to C#

When building applications with C#, it's often necessary to interact with a database to store, retrieve, or modify data. One of the most popular databases is SQL Server, and in this blog, we’ll guide you through the process of connecting your C# application to a SQL Server database. By the end, you’ll know how to interact with your database using basic operations like SELECT, INSERT, UPDATE, and DELETE.

What Do You Need?


• 1. SQL Server : The database server where your data is stored.

• 2. Visual Studio : The IDE where you write and run your C# application.

• 3. SQL Server Management Studio (SSMS) : A tool for managing your SQL Server database.

• 4. Connection String : A string that contains information required to connect to your SQL Server database, such as the server name, database name, username, and password.

Step 1 : Set Up SQL Server

• 1. Install SQL Server : Download and install SQL Server (Express Edition is free).

• 2. Create a Database : Use SQL Server Management Studio (SSMS) to create a new database. For example, let’s create a database named MyDatabase.

• 3. Create a Table : You can create a table within the database to store your data. For example, create a simple table called Users :

              
CREATE TABLE Users (
    Id INT PRIMARY KEY,
    Name NVARCHAR(100),
    Age INT
    );

Step 2 : Install SQL Server Client Library in C#

In your C# application, you need to install the SQL Server client library. This allows your application to communicate with SQL Server.

1. Open Visual Studio and create a new Console Application.

2. Open NuGet Package Manager by right-clicking on your project in the Solution Explorer and selecting Manage NuGet Packages.

3. Search for and install the package System.Data.SqlClient (or Microsoft.Data.SqlClient for newer versions).

Step 3 : Write C# Code to Connect to SQL Server

Now it’s time to write C# code that will connect to the SQL Server database. We will use SqlConnection to establish a connection.

                
using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        // Define the connection string
        string connectionString = 
        "Server=your_server_name;
        Database=MyDatabase;User 
        Id=your_username;
        Password=your_password;";
        
        // Create a connection to SQL Server
        using (SqlConnection connection 
        = new SqlConnection(connectionString))
        {
            try
            {
                // Open the connection
                connection.Open();
                Console.WriteLine
                ("Connected successfully!");
                
            //Perform database operations here
            }
            catch (Exception ex)
            {
Console.WriteLine("Error: " + ex.Message);
}
}
}
}

Replace the , , and with your actual SQL Server credentials. The is the key to establishing the connection between your C# application and SQL Server.

Step 4 : Execute SQL Queries

Once the connection is established, you can perform database operations like SELECT, INSERT, UPDATE, and DELETE.

Example 1: SELECT Query (Retrieve Data) 
    
string query = "SELECT * FROM Users";
SqlCommand cmd = new SqlCommand
(query, connection);

// Execute the query and get the result
SqlDataReader reader = cmd.ExecuteReader();
while (reader.Read())
{
    Console.WriteLine($"Id: {reader["Id"]},
    Name: {reader["Name"]}, 
    Age: {reader["Age"]}");
}
reader.Close();
Example 2: INSERT Query (Add Data) 
    
string insertQuery = "INSERT INTO Users 
  (Id, Name, Age) VALUES(1, 'John Doe', 30)";
SqlCommand insertCommand = new 
  SqlCommand(insertQuery, connection);

//Execute the query without returning data
insertCommand.ExecuteNonQuery();  
Console.WriteLine("inserted successfully!");
Example 3: UPDATE Query (Modify Data) 
    
string updateQuery = 
    "UPDATE Users SET Age = 31 WHERE Id = 1";
SqlCommand updateCommand = 
    new SqlCommand(updateQuery, connection);
    // Execute the query
updateCommand.ExecuteNonQuery();  
Console.WriteLine("updated successfully!");
Example 4: DELETE Query (Remove Data) 
    
string deleteQuery = 
    "DELETE FROM Users WHERE Id = 1";
SqlCommand deleteCommand = 
    new SqlCommand(deleteQuery, connection);
    // Execute the query
deleteCommand.ExecuteNonQuery();  
Console.WriteLine("deleted successfully!");

Step 5 : Close the Connection

Once you are done with the database operations, it’s important to close the connection to free up resources.

    
connection.Close();
Console.WriteLine("Connection closed.");

Conclusion

Interacting with SQL Server from a C# application is straightforward once you understand the basic steps :

• 1. Install the SQL Server client library.

• 2. Set up a SQL Server database and create tables.

• 3. Write code to establish a connection using SqlConnection.

• 4. Perform operations like SELECT, INSERT, UPDATE, and DELETE using SqlCommand.

• 5. Always close the connection when done.

With these steps, you can easily connect your C# application to SQL Server and begin working with data.
